How to make Terminal colorful

Mac OS Big Sur Version 11.1
Terminal profile "iceberg"as a default
Default shell "zsh" (as teacher said, it doesn't matter much for Mac user, my shell is "zsh"as default)

First to install

git clone --recursive https://github.com/sorin-ionescu/prezto.git "${ZDOTDIR:-$HOME}/.zprezto"

Back up if needed

mkdir zsh_backup mv .zlogin .zlogout .zprofile .zshenv .zshrc zsh_backup

Initial setting

setopt EXTENDED_GLOB for rcfile in "${ZDOTDIR:-$HOME}"/.zprezto/runcoms/^README.md(.N); do ln -s "$rcfile" "${ZDOTDIR:-$HOME}/.${rcfile:t}" done
